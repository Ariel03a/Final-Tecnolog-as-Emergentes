## Uso de Inteligencia Artificial

- **Herramienta usada:** GitHub, Copilot y React
- **Dónde fue útil:**
  - Generación de componentes (`Home.jsx`, `Catalogo.jsx`).
  - Generación de los datos de 'projects.js'.
  - Lógica de filtrado y ordenamiento.
  - Sugerencias de estilos con CSS.
  - Correciones de CSS.
- **Reflexión:**
  La IA permitió acelerar el proceso de desarrollo, mejorar la organización del código y aplicar buenas prácticas. Fue especialmente útil para estructurar componentes y refactorizar estilos de forma eficiente. No obstante, está claro que no está libre de polémicas y depender exclusivamente de su uso, puede crear un frankenstein (hablando de la creación de código) tomando los componenetes que sirven o adaptándolos y dejando un código que quizá nadie entienda al final. Por lo tanto, es importante saber qué hizo la IA y qué se podría cambiar para crear un codigo entendible.
